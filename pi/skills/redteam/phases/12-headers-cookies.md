# Phase 12: Headers, Cookies, and Side Channels

Audit HSTS, CSP, X-Frame-Options/frame-ancestors, X-Content-Type-Options, Referrer-Policy, Permissions-Policy, Cache-Control on sensitive endpoints, cookie Secure/HttpOnly/SameSite/domain/path, trace IDs, server banners, and debug headers. Severity is usually low/info unless cache or cookie scope exposes sensitive data.
