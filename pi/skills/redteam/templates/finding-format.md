# Finding Format Template

Required fields:
- ID and title
- Severity and rationale (CVSS if enough data)
- Category / CWE / OWASP mapping
- Affected scope (hosts/endpoints)
- Impact in plain language
- Minimal reproduction steps
- Evidence path(s) under `findings/` and `outputs/`
- Root cause if determinable
- Concrete remediation
- References
- Redaction / zero-retention note

Do not include bulk data, full tokens, full cookies, victim PII, or unverified scanner claims.
